# Conecta Sangue

**Descrição**  
Projeto desenvolvido como integração das disciplinas do primeiro semestre: Engenharia de Software, Lógica de Programação e Programação Web. O sistema tem como objetivo conectar doadores de sangue a pessoas necessitadas que precisam de transfusões, facilitando comunicação e localização de doadores compatíveis.

---

## Características principais
- Integração das aprendizagens do primeiro semestre (engenharia de software, lógica de programação, programação web).  
- Interface construída com **HTML**, **CSS** e **JavaScript** (front-end puro).  
- Cadastro de doadores e receptores (simulado via formulário — sem backend persistente neste estágio).   
- Exibição de lista de doadores com informações de contato (para fins de demonstração).  
- Validação de formulários usando JavaScript para garantir entrada de dados mínima (nome, tipo sanguíneo, contato).

---

## Tecnologias
- HTML5
- CSS3
- JavaScript 
---

## Objetivo do sistema
Conectar de forma simples doadores de sangue com pessoas que necessitam de transfusões, permitindo o registro de doadores e a busca por compatibilidade de tipo sanguíneo e proximidade (conceitual / para demonstração).
